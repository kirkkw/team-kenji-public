Welcome to ComPair

ComPair is a web interface that utilizes opinions and personal experiences in order to provide a platform in which users can learn about the differences between potential career fields.

The main function of this application is give the user a better sense of difference when it comes to two different career fields. Students, parents, and even professionals can benefit from using this application in order to see the pros and cons of two different paths.

If you are a registered user, please log in. If you are a new user, you may use the registration form to sign up.
